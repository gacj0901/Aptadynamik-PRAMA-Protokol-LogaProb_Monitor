# PRAMA Qualitative Phase Signature Protocol

This protocol adds an observer layer for qualitative phase-transition signatures in PRAMA Monitor sessions.

## Scope

The module evaluates qualitative signatures over observed PRAMA Monitor window series. The initial viability proxy is:

```text
viability = avg_rigidity - avg_uncertainty
```

For raw monitor windows this is computed per window as:

```text
viability = rigidity - uncertainty
```

## Signatures

### Discontinuity

Discontinuity is detected as a large adjacent jump in viability or rigidity.

### Hysteresis

Hysteresis is detected by comparing response branches during rising and falling pressure sweeps. Real sessions may not contain a clean pressure sweep, so the result is only meaningful when the session design includes pressure increase and release.

### Critical Slowing

Critical slowing is detected from delayed recovery after a large viability drop or from late increases in autocorrelation and variance.

### Structural Target Effect

Structural target effect compares peripheral prompts against constitutive prompts. Constitutive prompts are operationally approximated by keywords that target role, identity, rules, constraints, contradiction, or exact compliance.

## Synthetic Validation

Run:

```bash
python scripts/prama_phase_signature_runner.py --synthetic
```

Synthetic validation is instrument validation, not model evidence. It only confirms that the detectors respond to a known fold-like synthetic system and remain quiet for a smooth synthetic system.

## Real Session Analysis

Run:

```bash
python scripts/prama_phase_signature_runner.py --from-results results/diagnostic_001/
```

Model evidence begins only when `drive_system` is replaced by real PRAMA sessions, specifically `session_*_raw.json` files generated by PRAMA Monitor.

## Outputs

The runner writes:

- `phase_signatures.csv`
- `phase_report.md`
- `comparative_phase_summary.csv` when multiple models are present

The report must be read as qualitative diagnostics, not as a truth or factuality classifier.
